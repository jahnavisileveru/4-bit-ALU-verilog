\# 4-bit ALU Design using Verilog HDL



\##  Overview

This project implements a 4-bit Arithmetic Logic Unit (ALU) using Verilog HDL. 

The ALU performs basic arithmetic and logical operations based on control signals.



\##  Features

\- Addition (A + B)

\- Subtraction (A - B)

\- Bitwise AND

\- Bitwise OR

\- Bitwise XOR



\##  Design Details

\- 4-bit input operands: A and B

\- 3-bit select line to choose operation

\- Combinational logic design using `case` statement



\##  Testbench

A separate testbench (alu\_tb.v) is used to verify functionality of all operations.



\##  Tools \& Technologies

\- Verilog HDL

\- Simulation Tools (ModelSim / Cadence Xcelium / Vivado)



\## File Structure

\- `alu.v` → ALU design module  

\- `alu\_tb.v` → Testbench for simulation  

\- `README.md` → Project documentation
